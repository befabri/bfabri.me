## Description du projet

**Clash of Dev** est une compétition de développement front-end, opposant des développeurs à travers deux rounds intenses pour décrocher le titre de "Meilleur Dev".
L'événement est diffusé en direct sur Twitch et évalué par un jury de professionnels reconnus.

Les participants devaient intégrer des maquettes fournies sur Figma, créées spécifiquement pour l'événement.

**Jury :**
- **@[LLCoolChris_](https://x.com/LLCoolChris_)**, développeur renommé pour son expertise technique
- **[@maislina_](https://x.com/maislina_)**, Product Designer, créatrice des maquettes à intégrer

**Phases de la compétition :**
1. **La Piscine** (premier round) - Maquette dévoilée le 17 septembre 2024, projets remis le 23 septembre 2024
2. **La Finale** - Maquette reçue le 1er octobre 2024, projets remis le 8 octobre 2024

Les participants sont jugés selon un barème rigoureux, évaluant à la fois le design et la qualité du code.

### Critères d'évaluation de La Piscine :
- **Respect du design** : 6 points
- **Propreté du code** : 6 points  
  - Lighthouse score : 2 points  
  - Organisation du code : 2 points  
  - Sémantique HTML : 2 points
- **Responsive** : 2 points
- **Motion** : 3 points  
  - Originalité : 1,5 points  
  - Cohérence avec le design : 1,5 points
- **Easter Egg** : 3 points

### Critères d'évaluation de La Finale :
- **Respect du design** : 8 points
- **Propreté du code** : 3 points  
  - Fonctionnement / Core feature / Bugs : 2 points  
  - Mobile : 1 point
- **Motion** : 3 points  
  - Originalité : 1,5 points  
  - Cohérence avec le design : 1,5 points
- **Prise de risque** : 6 points

### Mes résultats :

**La Piscine :**  
- Design : 5,5 points  
- Développement : 6 points  
- Responsive : 1,5 points  
- Motion : 2,25 points  
- Easter Egg : 1 point  

**Total** : 16,25/20 - 4e place sur 50 participants

**La Finale :**  
- Code : 2,75 points  
- Design : 7,95 points  
- Motion : 1,75 points  
- Prise de risque : 3 points  

**Total** : 15,45/20 - 4e place sur 6 participants

## Développé avec

**Piscine:** AstroJs, TypeScript, PreactJs, Tailwind CSS

**Finale:** PreactJs, TypeScript, PreactJs/signals, Tailwind CSS
