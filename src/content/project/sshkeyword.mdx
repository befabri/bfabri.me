## Description du projet

{frontmatter.name} est un programme Go en CLI conçu pour pouvoir rapidement se connecter en SSH à partir d'un terminal.
Il permet donc aux utilisateurs de se connecter en utilisant des mots-clés qui ont été préalablement définis. Il y a aussi la possibilité de configurer une connexion SSH par défaut afin de se connecter sans mot-clé.

## Fonctionnalités Clés

- Assignez des mots-clés faciles à retenir à vos connexions SSH.
- Connectez-vous aux hôtes SSH en utilisant des mots-clés.
- Définissez une connexion SSH par défaut pour un accès instantané sans spécifier de mot-clé.
- Ajoutez de nouvelles connexions SSH avec un utilisateur, un port et des mots-clés.
- Modifiez les connexions existantes pour mettre à jour les détails si nécessaire.
- Supprimez facilement les connexions obsolètes.
- Listez toutes les connexions enregistrées avec leurs configurations.
- Disponible pour les systèmes Windows et Linux.
